# Hi, I'm Kyler 👋
I'm a Computer Engineering student at McMaster University, graduating Summer 2026!
I'm focused on building impactful autonomous systems with real-world applications. My current interests span robotics, reinforcement learning, sim-to-real transfer, and embodied AI.

### What I'm Working On
- **RL Navigation**: Training PPO policies for a Clearpath Jackal robot in NVIDIA Isaac Lab, studying how simulation fidelity affects sim-to-real transfer
- **AI Surveillance**: Multi-camera loitering detection system using NVIDIA DeepStream + PeopleNet on Jetson-class hardware with RTSP streaming and WebSocket feeds
- **Embedded Systems**: Real-time firmware development with PX4/NuttX and MAVLink on NXP FMU hardware
